# 项目说明

## 目标

本项目实现一个基于 ESP32 的局域网小喇叭：

- ESP32 连接 WiFi。
- ESP32 通过 MQTT 接收控制命令。
- PHP 提供本地网页和 HTTP API。
- 网页通过 PHP API 向 MQTT 发布命令。
- ESP32 通过 MAX98357A I2S 功放播放 MP3/WAV 网络音频。

## 当前版本

当前版本为 `1.0.0-local`，定位是本地局域网可用版本。

## 当前能力

- 播放 HTTP MP3/WAV URL。
- 暂停、恢复、停止。
- 设置音量。
- 播放本地测试音，用于排查 I2S 接线和功放问题。
- 设备上报 status 和 heartbeat，用于调试。

## 主要限制

- 不直接支持 HTTPS 音频流。
- 不直接支持 M4A 容器。
- 网页当前不依赖设备在线状态，只负责发送命令。

## 推荐调试顺序

1. 启动 Mosquitto。
2. 启动 PHP 内置服务器。
3. 烧录 ESP32 固件。
4. 串口确认 ESP32 已连接 WiFi 和 MQTT。
5. 网页点击“测试音”确认 I2S 功放有声音。
6. 再测试 HTTP MP3/WAV URL 播放。
